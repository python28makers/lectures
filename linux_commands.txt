ls - просмотр содержимого текущей директории
ls -a - просмотр скрытых файлов

абсолютный путь - полный путь(от домашней директории)
относительный путь - путь относительно местонахождения

. - ссылка на текущую директорию
.. - ссылка на директорию выше

touch название_файла - создание файла
mkdir название_директории - cоздание директории

cat название_файла - просмотр содержимого файла
cp название_файла название_копии_файла - копирование файла
echo 'Текст' - выводит указанный текст в консоль/терминал
nano название_файла - открыть текстовый редактор

> - перезапись вывода команды в файл
>> - дозапись вывода команды в файл
если файла не существует, то создает его
echo 'Hello world' > some_file

Выйти из nano:
ctrl + x -> Y -> Enter
